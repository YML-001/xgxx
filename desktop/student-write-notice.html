写信须知

提交至 学工信箱

您好！为使您的来信得到及时办理和明确答复，请在写信前仔细阅读以下内容：

  1. 请遵守国家法律法规和社会公德，如实反映情况，文明表达诉求。
  2. 请遵循「一事一信」原则，标题简明扼要，内容实事求是。
  3. 为便于核实与回访，建议使用真实姓名与联系方式；不愿公开的可选择「不公开」。
  4. 同一事项已办结后，请勿以相同事实和理由重复提交。

✓ 受理范围

  • 对学工部教育、管理、服务等方面的意见与建议
  • 学习、生活中遇到的困难与合理求助（资助、住宿、心理等）
  • 学工部各科室职责、政策、办事流程的咨询
  • 对学工部职责范围内事项的投诉与反映

✕ 不予受理

  • 非学工部职责范围内的事项（如教务、后勤、网络等）
  • 涉及 110、119、120 等紧急救助的事项
  • 涉及国家机密、个人隐私及违反公序良俗的事项
  • 应通过诉讼、仲裁、纪检监察等法定程序解决的事项
  • 已依法依规办结，仍以同一事实重复提出的事项
